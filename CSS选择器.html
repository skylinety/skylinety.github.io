兄弟选择器

CSS 常见的兄弟选择器包括 + ~
其中+ 是相邻兄弟选择器，~ 是通用兄弟选择器。
选择器前后的标签有有相同父标签。

相邻兄弟选择器 + 选择的是紧邻在前置标签后的第一个标签。
通用兄弟选择器命中的标签必须是(不一定是紧邻)前置标签后面的兄弟标签。

选择器 描述 同父 后置标签要求首个 选中标签
‘+’ 相邻 是 是 最多一个（非紧邻不选中）
‘~’ 通用 是 否 多个
